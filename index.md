## 🎮 О проекте

**EnderPirl Launcher** — это современный лаунчер для Minecraft, созданный с любовью к игре. Он сочетает в себе стильный дизайн в тематике Эндер-мира и мощный функционал.

### Ключевые особенности

- 🎮 Поддержка всех версий Minecraft от 1.8.9 до 1.21
- ⚙️ Встроенная установка Forge, Fabric и OptiFine
- 👻 Офлайн-режим и вход с паролем
- 💜 Стильный дизайн в тематике Эндер-мира
- 🚀 Быстрый запуск и оптимизация
- 🔄 Автоматическое обновление

## 📋 Системные требования

### 🟢 Минимальные (для лаунчера и Minecraft 1.8.9)

| Компонент | Требование |
|-----------|-------------|
| **ОС** | Windows 7 / 8 / 10 / 11 (32/64 bit) |
| **Процессор** | 1 ядро, 1.2 ГГц (Intel Atom, Celeron) |
| **ОЗУ** | 1 ГБ (512 МБ для лаунчера + 512 МБ для игры) |
| **Видеокарта** | Любая с OpenGL 2.1 |
| **Место на диске** | 200 МБ для лаунчера + 1-2 ГБ для Minecraft |
| **Java** | Java 8 (установится автоматически) |

### 🟡 Рекомендуемые (для комфортной игры)

| Компонент | Требование |
|-----------|-------------|
| **ОС** | Windows 10 / 11 (64-bit) |
| **Процессор** | 2 ядра, 2.0 ГГц (AMD Athlon II / Intel Core 2 Duo) |
| **ОЗУ** | 3-4 ГБ |
| **Видеокарта** | 1 ГБ VRAM, OpenGL 3.3+ (Radeon HD 7000 / GeForce 600) |
| **Место на диске** | 5 ГБ |
| **Java** | Java 8 или 17 (автовыбор) |
